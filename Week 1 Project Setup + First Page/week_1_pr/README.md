Week 1 — Django Foundations

📚 Overview

This week focuses on understanding the core structure of Django and how a basic web application works.

Main goal:
	•	Learn how Django connects URLs, views, and templates
	•	Build a multi-page website with shared layout and dynamic content

⸻

🧠 Topics Covered
	•	Django project & app structure
	•	URLs and routing
	•	Function-based views
	•	Templates (HTML rendering)
	•	Template inheritance (base.html)
	•	Template blocks ({% block %})
	•	Template includes ({% include %})
	•	Static files (CSS)
	•	Named URLs and {% url %}
	•	Dynamic URLs with parameters
	•	Loops and conditions in templates

⸻

🛠 Project Built

City Explorer

A multi-page website with:
	•	Home page
	•	About page
	•	Contact page
	•	Places listing page
	•	Dynamic place detail pages

⸻

⚙️ Features Implemented
	•	Reusable layout using base.html
	•	Header and footer components using includes
	•	Navigation with active link highlighting
	•	Dynamic routing for place detail pages
	•	Data passed from views to templates
	•	Looping through structured data (lists & dictionaries)
	•	Conditional rendering (ratings, status, etc.)
	•	Basic styling using CSS

⸻

🧩 Key Concepts Learned
	•	Separation of logic (views) and presentation (templates)
	•	Importance of named URLs over hardcoded paths
	•	Structuring data in Python for easier template rendering
	•	Reusability through template inheritance and partials

⸻

🚀 Result

At the end of Week 1, I can:
	•	Build a structured Django project
	•	Create dynamic multi-page websites
	•	Connect backend logic with frontend templates
	•	Apply basic UI styling and navigation logic
